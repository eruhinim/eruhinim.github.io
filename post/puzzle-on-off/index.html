On/Off puzzle

2025-06-14
puzzles

Here’s something to play around with.

Numbers are ordered in a circle. Each number can go be ON or OFF at any given time. All numbers are OFF in the beginning.

The goal is to turn all of the numbers ON.

There is only one possible action that you can do at each turn:

  1. Choose a number. That number turns ON.
  2. The two numbers opposite to it turn ON as well.
  3. If a number is already ON and is turned, it turns OFF.

For example if you turn on 1 - 5 and 4 with turn on as well, 7 - turns 7, 3 and 4 and so on.

Try it out. It’s fun.

import random

rules = {0: [0,3,4],
         1: [1,4,5],
         2: [2,5,6],
         3: [3,0,6],
         4: [4,0,1],
         5: [5,1,2],
         6: [6,2,3]}

while True:
    state = [False]*7
    r = list(range(7))
    random.shuffle(r)
    for i in r:
        ch = rules[i]
        state[ch[0]] = not state[ch[0]]
        state[ch[1]] = not state[ch[1]]
        state[ch[2]] = not state[ch[2]]
    if all(state):
        print(r)
        break

^Note for code: numbers are indexed from 0 to 6 since it’s more convenient.
